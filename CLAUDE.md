## Project Overview

家計簿アプリ（Kakeibo App） — a household budget tracking application.

## Git Rules

**Push to GitHub after every code change.**

Workflow for every change:
1. Stage the changed files: `git add <files>`
2. Commit with a meaningful message: `git commit -m "..."`
3. Push immediately: `git push`

Never leave committed changes un-pushed. Each logical change (bug fix, feature addition, refactor) should result in one push.
